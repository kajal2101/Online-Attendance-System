# Online Attendance System

## 📌 Project Overview
Online Attendance System is a web-based application developed to manage student attendance records digitally. It helps to store and manage attendance data easily.

## 🎯 Objective
The objective of this project is to make attendance management simple, fast and paperless.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## ✨ Features
- Student attendance management
- Digital attendance records
- User-friendly interface
- MySQL database connectivity
- Easy attendance tracking

## 📂 Project Files

- index.php
- style.css
- script.js
- connect.php
- screenshots

## ⚙️ How to Run Project

1. Install XAMPP
2. Start Apache and MySQL
3. Place project folder inside htdocs
4. Create database in phpMyAdmin
5. Open project in browser
